# Система управления банковскими картами

Backend-приложение для управления банковскими картами.

Приложение позволяет администраторам управлять пользователями и банковскими картами, а пользователям — просматривать свои карты, выполнять переводы между собственными картами и отправлять запросы на блокировку.

## Возможности

### Администратор

- создание банковских карт;
- просмотр всех карт;
- блокировка карт;
- активация карт;
- удаление карт;
- управление пользователями.

### Пользователь

- просмотр собственных карт;
- поиск и фильтрация карт;
- постраничная выдача;
- просмотр баланса;
- переводы между своими картами;
- создание запроса на блокировку карты.

# Безопасность

Реализовано:

- JWT-аутентификация;
- ролевая модель USER / ADMIN;
- разграничение доступа к API;
- шифрование номеров банковских карт;
- маскирование номера карты при выдаче:
  **** **** **** 1234

Пароли пользователей хранятся в виде BCrypt-хеша

# Запуск приложения

## Требования

Необходимо установить:

- Java 21
- Maven
- Docker

## Запуск через Docker Compose

## Настройка окружения

Создайте файл `.env` в корне проекта на основе примера:

```bash
cp .env.example .env
```

При необходимости измените значения переменных

Но CARD_ENCRYPTION_KEY должен содержать ключ длиной 32 символа

Запустить контейнеры:

```bash
docker compose up --build
```

После запуска приложение будет доступно:

Swagger UI: http://localhost:8080/swagger-ui.html
